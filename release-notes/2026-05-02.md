# OpenClaw v2026.05.02 版本發佈說明

[GitHub Release](https://github.com/openclaw/openclaw/releases/tag/v2026.5.2)

## ⚠️ 升級前必讀

### Breaking Changes 摘要

| 項目 | 影響 | 行動 |
|------|------|------|
| Thread Bindings 統一設定 | 舊的 `subagent/ACP thread-spawn` 拆分切換已由 `threadBindings.spawnSessions` 取代 | 執行 `openclaw doctor --fix` 自動遷移舊鍵值 ([#75943](https://github.com/openclaw/openclaw/pull/75943)) |
| Heartbeat 排程器行為變更 | 非 UTC `activeHours.timezone`（如 `Asia/Shanghai`）開始正確影響排程觸發時機 | 檢查 heartbeat 設定是否依賴舊的安靜時段行為（[#64016](https://github.com/openclaw/openclaw/pull/64016)、[#75487](https://github.com/openclaw/openclaw/issues/75487)） |
| Plugin runtime-deps 雜湊變更 | `plugin-runtime-deps` stage key 改用 OS-canonical `fs.realpathSync.native` 雜湊 | Windows/PM2 用戶升級後首次啟動會重新 stage bundled deps（[#75048](https://github.com/openclaw/openclaw/pull/75048)） |

### 新功能亮點

- **Plugin 全面 npm-first 外部化**：數十個 bundled plugins 移至 npm/ClawHub 獨立發佈，核心包大幅瘦身
- **Heartbeat 自我保護機制**：新增 centralized cooldown 防止 heartbeat 回饋環 runaway（`every: "30m"` 曾因 exec 事件觸發 ~10s 一次）
- **Tool Descriptor Planner**：平台級工具描述符快取，prompt 準備階段跳過 plugin runtime 載入
- **Voice Call per-number 路由**：撥入號碼獨立 greetings、response agents/models/prompts、TTS voice overrides
- **Session Store 全面非同步化**：transcript 讀取、標題/預覽水合、compaction 全部改為 bounded async IO

### ⭐⭐⭐ Heartbeat 排程器自我保護：Centralized Cooldown ([#64016](https://github.com/openclaw/openclaw/pull/64016))

- **用途**：新增 centralized cooldown 閘門，阻止 backgrounded `process.start` exit 通知自我觸發 heartbeat runaway；per-agent flood guard 限制 5 runs/60s；exec-event/notification/spawn/retry wakes 全部通過 cooldown；active-hours-aware scheduling 讓非 UTC timezone 正確影響排程

- **影響**：heartbeat 排程穩定，quiet-hours 不再浪費 tick，documented wake-now paths 保持即時
